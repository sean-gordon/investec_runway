Vision

Not just another dashboard.

Gordon is designed for the Investec Developer ecosystem. It's a complete C# .NET 8 backend with a Vue 3 frontend that turns raw transaction JSON into actionable actuarial runway modelling.

Most budgeting apps are reactive. They tell you what happened last month. Gordon is predictive. It uses your real historical burn rates to forecast your survival date — updated every time you swipe your card.

  • ✓ Real-time OAuth 2.0 transaction sync
  • ✓ TimescaleDB time-series data architecture
  • ✓ Actuarial Value-at-Risk runway modelling
  • ✓ Automated inter-account top-up transfers
  • ✓ Streaming AI responses (Gemini / Claude / Ollama)
01

Predictive Analytics

Custom actuarial models calculate your "Expected" and "Safe" runway days based on 24 months of historical volatility.

// Gordon Finance Engine — stack overview


runtime .NET 8 Web API + Background Workers

frontend Vue 3 (Options API) — single file, no build step

database TimescaleDB (PostgreSQL 16) — hypertables for time-series

orm Dapper — lightweight, direct SQL

auth JWT Bearer + rotating httpOnly refresh cookies

encryption AES-256 via .NET Data Protection for stored keys

ai Gemini / Claude / Ollama — configurable, user-keyed

messaging Telegram Bot API + Twilio WhatsApp + SMTP

deployment Docker Compose — single command, self-hosted

banking Investec Programmable Banking API (OAuth 2.0)
